# Movie Listing App 🎬

## 📌 Project Overview
A React-based movie listing app using the OMDB API to search and display movie details.

## 🚀 Features
- Search movies by title
- View movie details
- Responsive design

## 🛠️ Tech Stack
- React.js
- Axios (for API calls)
- React Router
